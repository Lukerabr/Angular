Instalar o Nodejs > www.nodejs.org
Instalar o Visual Studio Code > www.code.visualstudio.com
Abrir o terminal no Visual Studio Code através do atalho "ctrl + '"
Navegar até a pasta do projeto
Dentro desse diretório realizar a instalação do Angular, através do comando "npm install -g @angular/cli"
A partir disso será possivel a utilização de comando do Angular CLI
Execultar o comando "ng serve --open"
Realizar os testes
